# Fire TV Cast Changelog

## [Initial Release] - {PR_MERGE_DATE}

- Cast YouTube links to Fire TV via SmartTube
- Cast movies and shows to Fire TV via Stremio (with Cinemeta search)
- Optional projector Wake-on-LAN support
- HDMI-CEC fallback for projectors without network control
- Progress toasts: "Turning on projector…", "Waking Fire TV…"
